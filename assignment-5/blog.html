Important FAQ

What is the Document Object Model (DOM)?


The Document Object Model (DOM) is a programming interface for web documents, such as HTML or XML. It represents the structure of a web page in a hierarchical tree format, where every element on the page (such as paragraphs, headers, images, and links) is represented as a node. The DOM allows programming languages like JavaScript to access, modify, and interact with these elements dynamically.

How do you select an element from the DOM?

To select an element from the DOM, we can use:

1. getElementById(): Selects an element by its unique ID.
2. getElementsByClassName(): Selects all elements with a specific class name (returns a collection).
3. getElementsByTagName(): Selects all elements with a specific tag name (returns a collection).
4. querySelector(): Selects the first element that matches a given CSS selector.
5. querySelectorAll(): Selects all elements that match a given CSS selector (returns a NodeList).

What is event delegation in the context of DOM?

Event delegation in the context of the DOM is a technique where a single event listener is attached to a parent element to handle events triggered by its child elements. Instead of adding separate event listeners to each child, the parent listens for events from its children and processes them using event bubbling.

How do you manipulate an element's attributes and styles using DOM?

You can manipulate an element's attributes and styles using the DOM in the following ways:
Manipulating Attributes:

getAttribute(): Retrieves the value of an attribute.
setAttribute(): Sets or updates the value of an attribute.
removeAttribute(): Removes an attribute from an element.

Manipulating Styles:
Access the style property: Use the style property to directly change CSS styles (e.g., element.style.color = 'red'). Modify CSS classes: Add, remove, or toggle CSS classes using classList (e.g., element.classList.add('active')).
